    • Brendan Eich

      Brendan Eich, born 1961, is an American computer programmer and creator of the JavaScript scripting language. He is the chief technology officer at the Mozilla Corporation.

    • John Resig

      John Resig is an American computer programmer and entrepreneur, best known as the creator and lead developer of the jQuery JavaScript library.

    • Douglas Crockford

      Douglas Crockford is an American computer programmer and entrepreneur who is best known for his ongoing involvement in the development of the JavaScript language, for having popularized the data format JSON (JavaScript Object Notation), and for developing various JavaScript related tools such as JSLint and JSMin. He is currently a senior JavaScript architect at PayPal, and is also a writer and speaker on JavaScript, JSON, and related web technologies such as the Yahoo! User Interface Library (YUI).

    • Paul Irish

      Paul Irish is an American front-end web developer, and developer advocate for Google Chrome. He is also recognized as an expert in cutting edge Web Technologies like HTML5 and CSS3.
